# commonrpc
还在羡慕BAT，京东等公司的大流量的架构吗？让你的java系统引用解耦，互相独立,commonrpc 就可以办到。commonrpc 是一个以netty为基础，spring 自定义shcema为基础标签的rpc框架,不侵入任何业务代码，一个高性能分布式rpc框架，支持tcp协议,支持restful 方式的http协议
